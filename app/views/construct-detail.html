Description
创建一个新的GameObject, 以name为名字。
Creates a new game object, named name.
Transform在GameObject被创建的时候添加进去。没有参数的版本创建GameObject时会添加Transform组件，但不会添加其他组件。同样，只有一个字符串作为参数的构造器版本只创建了GameObject和Transform。第三个版本会将传递进来的组件数组添加进创建的GameObject。
Transform is always added to the GameObject that is being created. The creation of a GameObject with no script arguments will add the Transform but nothing else.the version with just a single string argument just adds this and the Transform.Finally, the third version allows the name to be specified but also components to be passed in as an array.
