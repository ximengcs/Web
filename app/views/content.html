GameObject
relation
Description
Unity场景(Scenes)中所有实体(独立存在)的基类。 注：在GameObject类中许多变量已经被移除了，比如C#中GameObject.renderer，它现在用GetComponent<Renderer>()来代替。 请参阅:Component.
Base class for all entities in Unity Scenes. Note: Many variables in the GameObject class have been removed. To access, for example GameObject.renderer in csharp use GetComponent<Renderer>() instead. See Also: Component.
